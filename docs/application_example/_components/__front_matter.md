## Application Example

The Ubuntu/Debian infrastructure can serve as Gateway applications, facilitating the seamless implementation of IoT Edge solutions. Furthermore, development using AWS IoT Greengrass supports remote monitoring through a Web-UI accessible on PCs and tablets.

![](application_example/images/AWS-IoT-Greengrass.png){ align=left width=120 }

* [EV Charging Station](https://renesas-wiki.atlassian.net/wiki/spaces/REN/pages/134053893/EV+Charging+Station+on+Ubuntu+Debian): OCPP protocol ver.2.0.1 implementation and web-socket communication.
* Home Automation (under prep.)): MQTT protocol communication for sensor data check & device control.
