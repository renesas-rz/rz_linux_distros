## Application Example

The Ubuntu/Debian infrastructure can serve as Gateway applications, facilitating the seamless implementation of IoT Edge solutions.
Furthermore, development using [AWS IoT Greengrass](https://aws.amazon.com/greengrass/){: target=_blank } supports remote monitoring through a Web-UI accessible on PCs and tablets.

![](application_example/images/AWS-IoT-Greengrass.png){ align=left width=160 }

* EV Charging Station: OCPP protocol implementation and web-socket communication.
	* [EV Charging Station for RZ/G2L EVK](https://renesas-wiki.atlassian.net/wiki/spaces/REN/pages/134053893/EV+Charging+Station+on+Ubuntu+Debian){: target=_blank }
	* [EV Charging Station for RZ/G3E EVK](https://renesas-wiki.atlassian.net/wiki/spaces/REN/pages/902561802/RZ+G3E+-+EV+Charging+Station+-+Ubuntu+Debian){: target=_blank }

* Home Automation: MQTT protocol communication for sensor data check & device control.
	* [Home Automation for RZ/G2L EVK](https://renesas-wiki.atlassian.net/wiki/spaces/REN/pages/899481601/RZ+G2L+-+Home+Automation+-+Ubuntu+Debian){: target=_blank }
	* Home Automation for RZ/G3E EVK (in progress, planned for Q4 2026)
